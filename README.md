# Truora SDK (deprecated)

> ⚠️ **This package is no longer maintained.**

The Truora SDK has been replaced by the **Truora Validations SDK**, which provides native document and face validation, on-device liveness detection, and a richer integration surface for iOS apps.

## Migrate to Truora Validations SDK

The recommended replacement is distributed via CocoaPods:

```ruby
pod 'TruoraValidationsSDK'
```

For the integration guide, see: <https://developer.truora.com/guides/validations_sdk_ios/>

## What happens to existing integrations?

- The `1.0.0` tag remains available; nothing has been removed from this repository.
- No further releases, fixes, or security patches will be published here.
- Existing integrations will continue to build, but will not receive new features.
